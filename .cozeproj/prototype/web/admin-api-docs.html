GET /api/products
获取商品列表

获取前台商品列表，支持分页、分类筛选和关键词搜索。返回商品基本信息，不含详细描述。

请求参数（Query）
参数名 类型 必填 说明
page integer 否 页码，默认 1
per_page integer 否 每页数量，默认 20，最大 50
category string 否 商品分类 slug，如 "dog-food"
q string 否 搜索关键词，模糊匹配商品名称
响应示例
{
  "code": 200,
  "message": "success",
  "data": {
    "total": 128,
    "page": 1,
    "per_page": 20,
    "items": [
      {
        "id": 1,
        "slug": "organic-dog-food-premium",
        "name": "有机犬粮-高端配方",
        "price": 299.00,
        "original_price": 399.00,
        "image": "https://images.unsplash.com/photo-1589924691994-45501a215ca5",
        "category": { "id": 1, "name": "犬粮", "slug": "dog-food" },
        "stock": 56,
        "status": "active"
      }
    ]
  }
}

slug string 必填 商品唯一标识符，如 "organic-dog-food-premium"

GET /api/orders
获取订单列表

获取当前用户的订单列表，支持按状态筛选和分页查询。需要认证。

请求参数（Query）
参数名 类型 必填 说明
status string 否 订单状态：pending / paid / shipped / completed / cancelled
page integer 否 页码，默认 1
per_page integer 否 每页数量，默认 10
响应示例
{
  "code": 200,
  "message": "success",
  "data": {
    "total": 5,
    "page": 1,
    "items": [
      {
        "id": "ORD-20240115-001",
        "status": "paid",
        "total_amount": 598.00,
        "items_count": 2,
        "created_at": "2024-01-15T10:30:00Z",
        "address": {
          "name": "张三",
          "phone": "138****5678",
          "city": "上海市"
        }
      }
    ]
  }
}
POST /api/orders
创建订单

基于购物车内容创建新订单。提交后将清空对应购物车项。需要认证。

请求体（JSON Body）
参数名 类型 必填 说明
address_id integer 必填 收货地址 ID
cart_item_ids array 否 指定购物车项 ID 列表，为空则提交全部购物车
remark string 否 订单备注，最大 200 字符
payment_method string 必填 支付方式：wechat / alipay / balance
响应示例
{
  "code": 201,
  "message": "订单创建成功",
  "data": {
    "id": "ORD-20240120-003",
    "status": "pending",
    "total_amount": 897.00,
    "items": [
      {
        "product_id": 1,
        "name": "有机犬粮-高端配方",
        "quantity": 2,
        "price": 299.00
      },
      {
        "product_id": 5,
        "name": "猫咪自动饮水机",
        "quantity": 1,
        "price": 299.00
      }
    ],
    "payment_url": "https://pay.animalsofa.com/checkout/xxx",
    "created_at": "2024-01-20T14:22:00Z"
  }
}

POST /api/cart/items
添加购物车商品

向购物车添加商品。如果商品已存在则累加数量。需要认证。

请求体（JSON Body）
参数名 类型 必填 说明
product_id integer 必填 商品 ID
sku_id integer 否 SKU ID，不传则使用默认 SKU
quantity integer 否 数量，默认 1，最大 99
响应示例
{
  "code": 200,
  "message": "已添加到购物车",
  "data": {
    "cart_item_id": 103,
    "product": {
      "id": 8,
      "name": "猫砂-豆腐猫砂 6L",
      "price": 59.90
    },
    "quantity": 3,
    "cart_total": 956.90
  }
}
