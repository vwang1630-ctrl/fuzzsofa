# 设计规范

## 品牌

- 品牌名：FUZZ SOFA
- 定位：动物灵感高端家具品牌
- 调性：奢华、手工、艺术、自然

## 配色

| 用途 | 色值 | 说明 |
|------|------|------|
| 主背景 | #0A0A0A | 深黑（前台页面） |
| 卡片背景 | #141414 | 深色卡片 |
| 主文本 | #F5F5F5 | 近白色 |
| 次文本 | #A0A0A0 | 灰色 |
| 品牌金 | #C9A96E | 金色强调 |
| 边框 | #2A2A2A | 深色边框 |

## 字体

- 标题：Playfair Display（衬线体）
- 正文：system-ui, -apple-system, sans-serif
- 中文：Noto Serif SC（衬线）

## 图片风格

- 暗调摄影风格
- 低饱和度
- 强调质感和光影
- 动物形态与家具融合

## 布局

- 全屏沉浸式
- 大量留白
- 大图展示
- 滚动叙事

## 禁忌

- 不要使用亮色背景
- 不要使用圆角过大的卡片
- 不要使用渐变按钮
- 不要使用 emoji 作为图标
